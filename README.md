# Intelligent Student Mental Health & Mood Tracker 🧠

An AI-driven, privacy-focused application that leverages state-of-the-art Natural Language Processing (NLP) to analyze student journal logs, mapping expressions to multi-class emotion vectors while proactively flagging academic burnout indicators.

## 🚀 Features
- **Local & Secure:** Built using a zero-configuration SQLite architecture to ensure all qualitative text data is persisted completely offline.
- **Transformer-Powered Core:** Utilizes a fine-tuned `DistilROBERTa` model optimized for 7-class human emotion distribution tracking.
- **Parallel Heuristic Validation:** Integrates syntax keyword validation engines to actively catch high stress thresholds in real-time.
- **Dynamic Analytics Dashboard:** Interactive data visualizations and historical tracking via Streamlit.
